# SmartPark

## Prerequisites
- Java 21

## Running
Run `./mvnw spring-boot:run` (Linux) or `.\mvnw spring-boot:run` (Windows) from the project root.

## Building
1. Run `./mvnw clean package`.
1. Run `java -jar target/smartpark-0.0.1-SNAPSHOT.jar` (change the version accordingly).

## Testing
- Run `./mvnw test`.
